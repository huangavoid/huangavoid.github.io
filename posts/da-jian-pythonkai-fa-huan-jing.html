Python

官方网站

下载源码，编译安装

wget http://python.org/ftp/python/2.7.6/Python-2.7.6.tar.xz
tar -xvf Python-2.7.6.tar.xz
./configure --prefix=/usr/local
make
make install 或 make altinstall

安装成功，查看版本

python -V
python2.7 -V

pip

官方文档

下载get-pip.py，执行安装脚本

wget https://bootstrap.pypa.io/get-pip.py
python2.7 get-pip.py

简明教程

pip install package             # 安装支持库
pip install --upgrade package   # 更新支持库
pip uninstall package           # 卸载支持库
pip list                        # 查看已安装的支持库
pip list --outdated
pip show --files package        # 查看支持库的对应文件

安装路径为：/usr/local/lib/python2.7/site-packages

virtualenv

官方文档

使用pip安装

pip2.7 install virtualenv

简明教程

virtualenv pylab                           # 创建虚拟环境
virtualenv -p python2.7 pylab              # 创建指定解析器的虚拟环境
virtualenv --system-site-packages pylab    # 创建继承系统第三方库的虚拟环境

source pylab/bin/activate                  # 启用虚拟环境
deactivate                                 # 退出虚拟环境

virtualenvwrapper

官方文档

使用pip安装

pip2.7 install virtualenvwrapper

简单配置

vim ~/.bashrc
export VIRTUALENVWRAPPER_PYTHON=/usr/local/bin/python2.7
source /usr/local/bin/virtualenvwrapper.sh

简明教程

mkvirtualenv pylab        # 创建虚拟环境
rmvirtualenv pylab        # 删除虚拟环境
lsvirtualenv              # 显示所有虚拟环境
workon pylab              # 启用虚拟环境
deactivate                # 退出虚拟环境

cdvirtualenv              # 跳至当前虚拟环境目录
cdsitepackages            # 跳至当前虚拟环境的site-packages目录
lssitepackages            # 显示当前虚拟环境的site-packages目录
